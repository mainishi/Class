To run my code:
1. Open IDLE 
2. Open lunar.py
3. Go to “Run” and then “Run Module”
4. The game will start by saying "Welcome to Lunar Lander" and the initial values for altitude, velocity, and fuel will print 
5. The user will be prompted "How much fuel would you like to burn?"
6. After entering a value and pressing “enter,” the current fuel, velocity and altitude will print
7. If the altitude is above 0, the user will be promoted again "How much fuel would you like to burn?" 
8. This cycle will continue until the altitude reaches 0
9. When altitude reaches zero, the program will say "Oh no! You lose! Unsafe landing" if velocity is greater than 10 m/s or will say "You win! Congratulations on successfully landing on the moon" if the velocity is less than 10 m/s
10. For either result, the user will then be promoted "Would you like to play again?"
11. If yes, the program will respond “Yay” and start over. If no, the program will say "Thank you for playing, goodbye"
